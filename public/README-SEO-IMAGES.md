# SEO Images Required

Please create and add the following optimized images to improve SEO:

## Required Images:

1. **og-image.png** (1200x630px)
   - Open Graph image for social media sharing
   - Should include your name, title, and branding
   - Use high contrast colors and readable fonts

2. **apple-touch-icon.png** (180x180px)
   - Apple touch icon for iOS devices
   - Should be your logo or profile picture
   - Use PNG format with transparent background if needed

3. **android-chrome-192x192.png** (192x192px)
   - Android Chrome icon
   - Should match your branding

4. **android-chrome-512x512.png** (512x512px)
   - High-resolution Android Chrome icon
   - Should match your branding

5. **favicon.ico** (32x32px, 16x16px multi-size)
   - Browser tab icon
   - Should be simple and recognizable at small sizes

## Tools for Creating Images:
- Canva (for og-image)
- Figma (for all images)
- Adobe Photoshop/Illustrator
- Online favicon generators
- Real Favicon Generator (realfavicongenerator.net)

## Tips:
- Use consistent branding across all images
- Ensure text is readable at different sizes
- Test images on different devices and platforms
- Optimize file sizes for faster loading
